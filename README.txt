First Run the "createdb.c" file - It will create account of admin with username as -admin and password as -admin


Then Run the "createAcc.c" file - It will create a file which will initialize account numbers as zero.

Now run the "server.c" file to start the server.

And then run the "client.c" file.

For creating a new user first login as admin and select the option to add new user.
